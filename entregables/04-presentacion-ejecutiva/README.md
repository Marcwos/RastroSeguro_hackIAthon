# 4. Presentación ejecutiva

Pitch del equipo (~10 min) para el jurado del hackIAthon 2026.

| Archivo | Formato | Uso |
|---------|---------|-----|
| `pitch.pdf` | PDF | **Entregable formal** |
| `pitch.md` | Markdown | Fuente editable |

## Contenido

1. Problema y oportunidad
2. Solución propuesta (RastroSeguro)
3. Demo funcional
4. Arquitectura y uso de IA
5. Impacto de negocio
6. Limitaciones y próximos pasos
7. FAQ del jurado

## Regenerar PDF

```bash
pip install fpdf2
py -3 scripts/export_pitch_pdf.py
cp presentation/pitch.pdf entregables/04-presentacion-ejecutiva/pitch.pdf
```
